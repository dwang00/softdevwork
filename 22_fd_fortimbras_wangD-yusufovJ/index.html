Layout

Back to Top

Layout in Foundation 6 is done using containers, which are adaptive elements that change in size as the viewport changes. The main component of layout in Foundation 6 is the X-Y grid, which allows the user to set columns and rows with responsive widths, add gutters and padding, and allocate space on the screen as they see fit.

Buttons, switches, and sliders (JavaScript based) are three of the options for controls that Foundation offers.
